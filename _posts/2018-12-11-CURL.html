CURL

 

Can be used for testing end points downloading files

 

for grabbing a url

 

{{site.cbs}}curl <url>{{site.cbe}}

 

to output to a file

 

{{site.cbs}}curl -o <fileName> <url> {{site.cbe}}

 

to include header information

 

{{site.cbs}}curl -i <url> {{site.cbe}}

 

to just get head

 

-I or --head

 

to download file with curl

 

{{site.cbs}}curl -O <url> {{site.cbe}}

 

Post request

is the post request url

 

{{site.cbs}}curl -d "title=Hello&body=Hello World" <url> {{site.cbe}}

 

PUT Request

{{site.cbs}}curl -X PUT -d "title=Hello" <url> {{site.cbe}}

 

Delete Request

{{site.cbs}}curl -X DELETE <url> {{site.cbe}}

 

For secure routes

 

{{site.cbs}}curl -u <username>:<password> <url> {{site.cbe}}

 

Follow redirection

{{site.cbs}}curl -L <url> {{site.cbe}}

 

FTP Protocol

 

Uploading a file using CURL

 

curl -u test@traversymedia.com:12345 -T ftp://ftp.traversymedia.com

 

curl -u : -T

Download files by ftp

 

curl -u : -O /
