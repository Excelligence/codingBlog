Introduction

textContent can be used to change text of an existing element. This will replace existing content with new content. createTextNode can be used to add to existing text content.

HTML {{site.cbs}} <p id='para'>Lorem ipsum dolor sit amet laboriosam dicta fugit? </p> {{site.cbe}}

JavaScript {{site.cbs}} const p =document.getElementById('para'); {{site.cbe}}

This will append. {{site.cbs}} p.append(document.createTextNode(' This created text node text')) {{site.cbe}}

This will replace with new text {{site.cbs}} p.textContent='this is the changed text'; {{site.cbe}}
