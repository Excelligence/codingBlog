Introduction

This short post explores the key difference between textContent and createTextNode properties in javascript. textContent is used to change text of an existing element. This will replace existing content with new content. createTextNode is used to add to existing text content.

I have explained it with the HTML and JS below.
HTML

We have a paragraph element with some text.

{{site.html}} <p id='para'>Lorem ipsum dolor sit amet laboriosam dicta fugit? </p> {{site.cbe}}

JavaScript

Here we are grabbing the element with id 'para' to a variable p

{{site.js}} const p =document.getElementById('para'); {{site.cbe}}

In next step we create a text node and append that to the paragraph element. This will add to the existing content of paragraph element. It will not replace it.

{{site.js}} p.append(document.createTextNode(' This created text node text')) {{site.cbe}}

In contrast if we use textContent as demonstrated below the existing text will be replaced with new text.

{{site.js}} p.textContent='this is the changed text'; {{site.cbe}}

You can verify each of the above output by logging them to console.
