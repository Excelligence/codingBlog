#### Two ways to change base font size for rem unit.

+ First Method

Add this to your css

```
html{
    font-size:10px;
}
```
Change the font-size from 10px to whatever you require.

+ Second Method

Add this to your css

```
:root{
    font-size:10px;
}
```
Change the font-size from 10px to whatever you require.
